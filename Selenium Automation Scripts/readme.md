# Selenium Automation Script

This folder includes **Selenium automation script** designed to perform web tasks, such as searching for products, interacting with items, and adding them to the cart on platforms like Amazon. The script showcase **Python-based web automation using Selenium**.

1. Open a terminal or command prompt in this folder.
2. Run the script:

```bash
python amazon_add_to_cart.py

```
3. The script will:

   - Open Amazon in a browser
   - Search for a product (e.g., "laptop")
   - Click the first product image
   - Click "Add to Cart"
   - Wait a few seconds before closing the browser
